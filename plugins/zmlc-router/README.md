# ZMLC Codex Plugin

This plugin exposes verified deterministic routing and compact task compilation to
Codex. Unsupported work returns directly to the active Codex model. It does not start
a local model, call another remote model, request an API key, or persist prompt text.

Use the platform-specific standalone release archive for a Python-free installation.
The repository source bundle uses `runtime/zmlc_router.pyz` for development.
